SKT War Result
===============================

version number: 1.0
author: yumenoshima

Overview
--------

Lists war results for Japanese servers in Lineage Mobile game.

Installation / Usage
--------------------

To install use pip:

    $ pip install skt-war-result


Or clone the repo:

    $ git clone https://github.com/yumenoshima/skt-war-result.git
    $ python setup.py install
